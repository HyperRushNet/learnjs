Usage:

document.addEventListener("eventType", handler)

Handler can be:

() => { /* code */ } function() { /* code */ }

Use cases: Responding to user actions, form validation, interactive UI, animations, logging events.

Simple Fetch

async function getData(url, type){ let fetched = await fetch(url); let res = await fetched[type](); // type = "text" or "json" return res; } // Example: getData("https://hyperrushnet.github.io/assets/json/games.json", "json") .then(data => console.log(data));

Use cases: Loading data from APIs, fetching JSON or text, dynamic content updates, AJAX-like behavior.

Wait helper function

async function wait(ms){ return new Promise(resolve => setTimeout(resolve, ms)) } // Use example console.log("console.log nr1") await wait(2000) // waits 2 seconds. console.log("console.log nr2")

Use cases: Waiting before action

Classes

// Define a class class Person { constructor(name, age) { this.name = name; // Property this.age = age; // Property } // Method greet() { console.log(`Hi, I'm ${this.name} and I'm ${this.age} years old.`); } } // Create instances const alice = new Person("Alice", 25); const bob = new Person("Bob", 30); // Call methods alice.greet(); // Hi, I'm Alice and I'm 25 years old. bob.greet(); // Hi, I'm Bob and I'm 30 years old.

Use cases: Structuring data, reusable methods, ...

Classes in classes

class outerClass { static outerFunction(p1, p2){ return p1 + ", " + p2 } static innerClass = class { static innerFunction(p1,p2){ return p1 + p2 } } } console.log(outerClass.outerFunction(1,2)) console.log(outerClass.innerClass.innerFunction(1,2))

Use cases: Structuring data even more.
